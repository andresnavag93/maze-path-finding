# Maze Escape

- A 2D game where kong (player) needs to solve a maze from a start point to an end point.
- Exists a hint button that shows a bananas path from the player position to the destination (using A* pathfinding)

### Technologies / Tools

- Unity 3D

### System requirements

- Unity ^v2020.3.21F1
